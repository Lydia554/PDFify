API Integration Guide (Developer Mode)

Learn how to use our PDF Generator API. This guide walks you through step-by-step instructions for generating beautiful, branded PDFs from your data.

üõçÔ∏è What Can You Do With It?

  • Generate product information sheets directly from your data
  • Create downloadable invoices for orders
  • Print shipping labels or packing slips
  • Design custom PDFs for gift cards or marketing inserts

‚öôÔ∏è How It Works

  1. Prepare Your JSON or Raw HTML

    You can provide structured JSON data or raw HTML content to generate a PDF.

  2. Upload to PDF Generator

    Submit your data or HTML through our interface. JSON will be parsed with the selected template, while HTML is directly rendered.

  3. Choose or Customize Template

    Select a built-in template or design your layout with HTML and CSS.

  4. Download the Result

    Click 'Generate' and your file will be ready to download.

  5. Each successful POST that returns a PDF counts toward your usage.

üì¢ Note for Developers: Handling the PDF Response

Our API returns the full PDF file data immediately in the response. Your application is responsible for processing this data to download or save the PDF file.

In Postman: After making the POST request, switch to the Body tab of the response, click Save Response (floppy disk icon), and save the file with a .pdf extension.

In your code: Handle the API response as a binary Blob (JavaScript) or save the raw response stream to disk (server-side), then trigger a file download or store the PDF as needed.

This approach gives you full control over how and when the PDF is saved or shown to your users.

üì¶ Packing Slip

fetch("https://pdf-api.portfolio.lidija-jokic.com/api/generate-packing-slip", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
  },
  body: JSON.stringify({
    data: {
      orderId: "98765",
      customerName: "John Smith",
      shippingAddress: "1234 Maple Street, Springfield, USA",
      date: "2025-05-14",
      items: [
        { name: "Yoga Mat", sku: "YM-001", quantity: 1 },
        { name: "Dumbbells Set", sku: "DB-002", quantity: 2 },
        { name: "Water Bottle", sku: "WB-003", quantity: 3 }
      ]
    }
  })
})
.then(response => response.blob())
.then(blob => {
  const url = window.URL.createObjectURL(blob);
  const a = document.createElement("a");
  a.href = url;
  a.download = "packing-slip.pdf";
  document.body.appendChild(a);
  a.click();
  a.remove();
});
    
curl -X POST "https://pdf-api.portfolio.lidija-jokic.com/api/generate-packing-slip" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "data": {
      "orderId": "98765",
      "customerName": "John Smith",
      "shippingAddress": "1234 Maple Street, Springfield, USA",
      "date": "2025-05-14",
      "items": [
        { "name": "Yoga Mat", "sku": "YM-001", "quantity": 1 },
        { "name": "Dumbbells Set", "sku": "DB-002", "quantity": 2 },
        { "name": "Water Bottle", "sku": "WB-003", "quantity": 3 }
      ]
    }
  }'
