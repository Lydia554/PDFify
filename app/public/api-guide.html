API Integration Guide (Developer Mode)

Learn how to use our PDF Generator API. This guide walks you through step-by-step instructions for generating beautiful, branded PDFs from your data.

🛍️ What Can You Do With It?

  • Generate product information sheets directly from your data
  • Create downloadable invoices for orders
  • Print shipping labels or packing slips
  • Design custom PDFs for gift cards or marketing inserts

⚙️ How It Works

  1. Prepare Your JSON or Raw HTML

    You can provide structured JSON data or raw HTML content to generate a PDF.

  2. Upload to PDF Generator

    Submit your data or HTML through our interface. JSON will be parsed with the selected template, while HTML is directly rendered.

  3. Choose or Customize Template

    Select a built-in template or design your layout with HTML and CSS.

  4. Download the Result

    Click 'Generate' and your file will be ready to download.

🔐 Developer Premium Benefits

  • Up to 1000 PDF pages/month
  • Remove or customize the watermark/logo
  • Inject or remove custom charts via code
